# 🔐 Lua Password Generator

A simple terminal-based password generator written in Lua.
It generates random passwords using letters, numbers, and special characters.

## Features

- Random password generation
- Custom password length
- Supports letters, numbers, and symbols
- Written in pure Lua
- Simple CLI interaction
- Easy to modify and extend

## Example

```text
How many characters do you wish your password to have?
12

Your new password was generated:
A9@kL2#pQ!7x
```

## Usage

Run the script using Lua:
```lua
lua src/main.lua 
```

## Requirements

- Lua 5.1 or higher
